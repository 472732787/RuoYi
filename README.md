## 平台简介

 基于RuoYi平台 集成rocketmq控制台  [ruoyi-rocketmq]
 
 配置文件：application-rocketmq.yml           
 
 参考：
 
 若依官网 http://doc.ruoyi.vip/
 
 RocketMQ-Console :    https://github.com/apache/rocketmq-externals/tree/master/rocketmq-console
